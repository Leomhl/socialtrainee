Instruções do app

Menu

  • Menu Principal: É no menu principal que a navegação pelo aplicativo será feita, por meio dele.

  • Jogar: O botão jogar leva para a tela de seleção de jogos onde poderá ser feita a escolha da modalidade e dos jogos desejados.

  • Instruções: O botão intruções leva para a tela onde existem as instruções de uso para o software (esta que você lê neste instante).

  • Sobre: O botão sobre contém todas as informações sobre o aplicativo, algumas informações de como foi desenvolvido, quem foram os responsáveis e etc.

  • Sair: O botão sair encerra a aplicação.

Atalhos

  • Botão retornar do celular: Para o caso dos aparelhos android e windows phone, os mesmos cotam com um botão de retornar para a página anterior. Este recurso está habilitado no Social Trainee para facilitar o retorno para a página anterior.

  • Botão home: O botão home, presente em todas as telas deste aplicativo direciona-lhe para a tela do menu principal para que alguma de suas opções possam ser usadas.
