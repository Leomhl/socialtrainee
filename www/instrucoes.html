Instruções do app





Menu

  • Menu Principal: É no menu principal que a navegação pelo aplicativo será feita. Ele pode ser acessado no botão localizado na barra superior esquerda da tela.

  • Jogar: O botão jogar leva para a tela de seleção de jogos onde poderá ser feita a escolha da modalidade e dos jogos desejados.

  • Instruções: O botão intruções leva para a tela onde existem as instruções de uso para o software (esta que você lê neste instante).

  • Informações: A tela de informações mostra os dados detalhados do usuário do aplicativo como: nome cadastrado, email e pontuação nos jogos organizada pela área de cada jogo (adm, segurança do trabalho e logística). Também há um botão que zera todos os pontos dos jogos tornando possível retornar a jogar partindo da pergunta 1.

  • Configurações: Na tela de configurações é possível inserir novas perguntas nos jogos ou excluir. Para cadastrar, basta informar a pergunta, as 4 opções de resposta, selecionar qual é a resposta certa e qual é a modalidade de jogo que aquela pergunta pertencerá. Para excluir o processo é mais simples, basta informar qual é a modalidade de jogo e selecionar a pergunta a ser excluída.

  • Sobre: O botão sobre leva para a tela que contém todas as informações sobre o aplicativo, algumas informações de como foi desenvolvido, quem foram os responsáveis e etc.

  • Sair: O botão sair encerra a aplicação.

Jogos

Para jogar, vá na tela de jogos, selecione a modalidade, estude as matérias propostas... Ao acionar o botão "Vamos jogar!" a tela de jogo será carregada e as questões serão carregadas para que o jogo comesse. Para cada questão respondida corretamente serão acrescidos 10 pontos. Para cada questão errada, serão decrementados 15 pontos. Quando o usuário quiser sair do jogo, basta ir ao menu e selecionar a opção "Terminar Jogo", ele pedirá uma confirmação e abrirá a tela principal.

Atalhos

Botão retornar do celular: Para o caso dos aparelhos android e windows phone, os mesmos contam com um botão de retornar para a página anterior. Este recurso está habilitado no Social Trainee para facilitar o retorno para a página anterior.
